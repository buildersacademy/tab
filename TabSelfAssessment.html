How to open and answer questions with Adobe Acrobat Reader
1. Navigate to the location of the file then right-click and select "Open with Adobe Acrobat Reader DC"

2. Once opened, click on the unit you want to apply for RPL on the "Bookmarks" panel (left-side of the window)

3. Please review the list of units and select the units that you want to apply for RPL then write an answer for each questions

How to save PDF File - Important - We recommed you to save the file regularly
1. To save the file, click "File" then select "Save As" (On the top bar) or Press and hold combination CTRL + SHIFT + S on your keyboard (For Windows) - ⇧⌘S (For Mac)

 or  (Windows)  -  (Mac)

2. "Save As PDF" window will open, select a location then click "Save". Please note, everytime you save the file, the same window will open, if you select the same location to save it will as if you want to replace the existing file - click "Yes"

(We recommed you to save the file regularly) 

How to upload and attach completed file
1.Click "Choose File" (At the end of this window)

2. A new window will show up called "My Documents". Click "Upload" (Top left of the window)

3. A File Browser will show up. Navigate or browse to the document you want to upload, select then click "Open"

4. Please wait while your document is being uploaded. 

5. Once the document is attached successfully, "My Documents" window will be closed and you will see the question again.

6. You'll see a list of documents you have uploaded and attached. 

Once completed, please click "Submit"
